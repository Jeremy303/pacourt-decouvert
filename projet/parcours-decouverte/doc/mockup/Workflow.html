L'Accueil (choisir)

Dans cette config, il y as la sidebar, la sidebar peut contenir différents mot clée pour indexée directement l'organisme en question, on peu aussi ce servir d'un input date, pour encore filtrée.. Cette page est volontairement réduite.
L'ajax peut être appliquée. Bémol : Le responsive sa vas être l'éclate.

L'autre accueil est un peu plus complète mais n'as de sidebar, toutefois ajax peut être calé au niveau des thead du tableau, il suffirais d'imaginais que le visiteur clique sur Date, pour mettre en avant les premières date. Ou de cliquée sur Organisme pour remontée chaque organisme par ordre alphabétique. L'ajax peut être intereressant si l'on fixe une limite au nombre de session, par exemple max 10 instances de session visible, l'ajax permettrait de mettre les dates ou les organismes a jour sans rafraichissement de page.
